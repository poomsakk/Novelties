## Getting start
1. install npm packages in frontend folder<br />
   ```
   cd frontend
   npm install
   ```
2. install npm packages in backend folder<br />
   ```
   cd backend
   npm install
   ```
3. run npm in frontend folder<br />
   ```
   npm start
   ```
4. run npm in backend folder<br />
   ```
   npm start
   ```
